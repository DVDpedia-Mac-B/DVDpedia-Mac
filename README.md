## Overview

DVDpedia Professional Suite is a complete media collection platform that serves serious collectors with the export flexibility, statistics, and cross-device access needed for managing large, well-curated film libraries.

Export and sharing produces collection lists in HTML, CSV, and XML formats for sharing with other collectors, publishing online, and importing into other cataloging systems. The HTML export creates browsable web pages with cover art and movie details. Collection statistics show library composition by genre, decade, format, and rating.

iCloud sync keeps the collection database current across all Apple devices, enabling collection browsing and lookup from iPhone and iPad alongside the full desktop application. The mobile access means that collection information is available while shopping for new additions or visiting friends' collections.

Wish list management tracks movies you want to acquire with price alerts and availability notifications from online retailers. The wish list integrates with the main collection, preventing duplicate purchases and tracking acquisition progress toward collection goals.

DVDpedia Professional Suite is the collection management platform for serious film collectors who need comprehensive cataloging, cross-device access, and collection analytics that support intentional, well-curated movie library building.

## Performance & Integration

DVDpedia is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while DVDpedia processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
